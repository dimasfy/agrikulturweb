Selamat Datang!

Selamat datang di sistem kontrol dan monitoring greenhouse kami! Kami menghadirkan teknologi untuk memastikan pencahayaan, kondisi udara, dan level air yang optimal bagi tanaman Anda. Dengan sensor BH1750 untuk cahaya, sensor DHT untuk suhu, serta sensor LDR dan laser KY008 untuk level air, semua terpantau dengan baik. Sensor FSR juga siap mendeteksi kapan hasil panen siap dikirim. Data dipantau secara real-time dan disimpan di database lokal. Sistem ini terintegrasi dengan protokol MQTT untuk efisiensi maksimal. Nikmati kemudahan dan keandalan teknologi ini untuk meningkatkan hasil panen Anda. Selamat berkebun!

Deskripsi Sistem

Selamat datang di sistem kontrol dan monitoring greenhouse kami! Sistem ini dirancang untuk menjaga kondisi optimal bagi tanaman dengan teknologi yang terintegrasi. Dengan sensor BH1750, intensitas cahaya diukur dan dikendalikan untuk memastikan pencahayaan yang sesuai. Sensor DHT memonitor suhu, mengaktifkan kipas jika diperlukan untuk menjaga suhu ideal. Kombinasi sensor LDR dan laser KY008 digunakan untuk mendeteksi level air, memastikan tanaman selalu mendapatkan air yang cukup.

Selain itu, sensor FSR mendeteksi tekanan hasil panen, memberi tahu kapan saatnya panen dikirim. Semua data dikirim secara real-time ke dashboard dan disimpan di database lokal untuk analisis lebih lanjut. Sistem ini terintegrasi dengan sistem air dan distribusi menggunakan protokol MQTT untuk efisiensi maksimal. Penempatan sensor dan aktuator yang strategis, serta kabel yang terorganisir rapi, memudahkan perawatan dan memastikan performa optimal.
